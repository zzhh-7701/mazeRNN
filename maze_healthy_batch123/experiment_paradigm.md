# 迷宫寻宝实验文档

## 实验介绍

本实验是一个迷宫寻宝任务。被试需要在每个试次中控制一个外星人在迷宫中移动，并到达指定目标。

整个实验围绕一个迷宫结构展开。对于每一个被试，迷宫结构在不同任务、不同试次、不同实验天数之间保持不变。实验重点考察被试是否能够逐渐记忆迷宫结构，并在后续任务中利用该记忆进行寻路、规划和重新规划。

不同实验批次之间存在范式调整。目前共有 7 个实验批次，可分为三种设计版本：

- 第 1-4 批：原始实验设计。
- 第 5-6 批：在第二天最后新增“重新规划” (replan) 任务。
- 第 7 批：第一天任务整体改为隐藏迷宫设计，第二天保持保持第 5-6 批设计。

## 第 1-4 批实验设计

第 1-4 批实验第一天包含 4 个任务，第二天继续完成 Task4 的后续 block。

### Day 1

#### Task1: Fixed Start and Goal

- 51 试次
- 被试可见 agent 周围上下左右的可移动道路
- 采用 mini-block 设计，每 3 个 试次 为一组，每个 mini-block 内起点和目标都固定

#### Task2: Fixed Goal, Changing Start

- 50 试次
- 被试可见 agent 周围上下左右的可移动道路
- 采用 mini-block 设计，每 5 个 试次 为一组，每个 mini-block 中目标固定，起点发生变化

#### Task3: Changing Start and Goal with Local Visibility

- 50 试次
- 试次间起点和目标都发生变化
- 被试可见 agent 周围上下左右的可移动道路

#### Task4 Day 1 Block 1: Changing Start and Goal with Hidden Maze

- 50 试次
- 试次间起点和目标都发生变化
- 被试看不到迷宫中的可移动道路

### Day 2

#### Task4 Day 2 Blocks 2-4: Changing Start and Goal with Hidden Maze

- 每个 block 包含 50 试次
- 试次间起点和目标都发生变化
- 被试看不到完整迷宫中的可移动道路

## 第 5-6 批实验设计

第 5-6 批的第一天和第二天前半部分保持原始设计，主要变化是在第二天最后新增 Task5 replan 任务。

### Task5: Plan and Replan

- 50 试次
- 试次间起点和目标都发生变化
- 被试看不到完整迷宫中的可移动道路
- 每个试次包含两个连续阶段：plan 阶段和 replan 阶段。

#### Plan 阶段

- Plan 阶段与 Task4 相同
- 被试从当前试次的起点移动到目标

#### Replan 阶段

- 当被试到达目标后，当前试次内的迷宫结构发生一次局部变化，变化形式为某一条路被 block 或 unblock，展示变化后，该变化会隐藏
- 被试需要在变化后的迷宫中从目标返回原起点
- 因此，一个 Task5试次中被试需要完成两段路径：
  - 起点到目标
  - 目标返回起点

Task5 中的迷宫结构变化只局限于当前试次内。
